# learncesium



* 标绘
  * 操作
    * 增（创建新的标绘）
    * 删（清除已有标绘）
    * 改（编辑已有标绘）
    * 查（脚本绘制标绘）
  * 事项
    * 绘制提示
    * 绘制过程可停止绘制
    * 所有标绘互斥
  * 类型
    * 点
    * 线：弧线、曲线、折线、自由线
    * 面：圆、椭圆、弓形、扇形、矩形、曲线面、多边形、自由面、聚集地、钳击、直箭头、细直箭头、突击方向、进攻方向、进攻方向（尾）、分队战斗行动、分队战斗行动（尾）
* 勘测
  * 操作
    * 测距（含周长）
    * 面积
    * ...
* 计算
  * 类型
    * 声传播损失、环境噪声、水声混响、作用距离（探测距离）、探测概率、态势（数据可视化）
    * 声速梯度、声跃变层、声速剖面、声道、会聚区、声影区、海底反弹区、直达波区
* 数据可视化
  * 要素类型
    * 温盐密（温度、盐度、海水密度）
    * 风浪流（低空风、高空风、海浪、海流）
    * 压湿水（气压、压强、湿度、降水）
    * 重磁冰（重力、磁力、电磁、积冰）
    * 底质、地形、AIS
  * 可视化类型
    * 热力图
    * 马赛克图
    * 等值线
    * 等值面
    * 凤羽图
    * 箭头
    * 粒子流
  * 维度
    * 时间
    * 深度 / 高度
  * 图例
